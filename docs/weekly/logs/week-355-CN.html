【2017年11月17日】MIUI 第355周发布公告及更新日志

论坛链接：【2017年11月17日】MIUI 第355周发布公告及更新日志

社区链接：【2017年11月17日】MIUI 第355周发布公告及更新日志


▍发布时间

MIUI9 开发版 7.11.16 11:00 发布


▍升级说明

  1. MIUI 8开发版可进入“设置→我的设备→MIUI版本”在线升级
  2. MIUI 6/7开发版以及 MIUI 8稳定版用户，建议通过线刷升级至 MIUI 9开发版，教程请见下方“刷机指南”
  3. 英文论坛同步更新英文版升级日志
  4. 因近期开发团队集中精力优化系统体验，更新日志可能会较少，望大家理解！

▍重要说明

  1. 快如闪电：MIUI 9稳定版正在开放升级中，已支持18款小米手机
  2. 小米手机5X 因修复电信卡运营商信号弱导致无法挂断电话的问题，为避免影响日常使用，公测暂停发布
  3. 红米手机Pro、红米Note 4、红米Note 4X-MTK 已恢复7.11.16公测版本推送

▍刷机指南

  1. MIUI 8/MIUI 9开发版可点击“设置”→“我的设备”→“MIUI版本”在线升级，升级前务必备份重要数据
  2. MIUI 6/MIUI 7开发版以及 MIUI 8稳定版用户版本用户建议通过线刷升级，线刷会清除全部数据
  3. 开发版/稳定版相互刷机时，需备份好重要数据，再通过线刷方式清除所有数据进行刷机，防止出现异常问题
  4. 新机型有防刷机保护，线刷前请先进行 Bootloader 解锁
