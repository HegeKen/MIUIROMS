MIUI 第170周发布公告及更新日志

原链接：【2013年12月27日】MIUI 第170周发布公告及更新日志（附视频）

▍发布时间

MIUI V5 3.12.27 已发布！


▍升级说明

  1. MIUI V5开发版可以直接OTA升级（方法：系统更新 - 检查更新 - 选择最新版本升级）
  2. 较老版本的MIUI可通过刷V5最新完整包快速升级（方法：小米手机可通过系统更新 - Menu - 选择安装包进行升级，第三方机型需手动进入Recovery选择完整包进行升级）
  3. 跨版本进行升级时（如稳定版刷开发版）尽量清除所有数据，以避免因数据冲突出现个例问题；清除数据前务必主动备份数据，以免丢失（方法：系统与安全 - 备份 - 新建备份）
  4. 其他品牌手机首次刷MIUI ROM 请严格按照刷机教程进行刷机

▍更新日志


【本周推荐】

新增 支持赠送主题

新增 视频支持离线下载（目前仅支持部分源）

新增 云相册支持视频同步

新增 文件管理器支持访问samba网络文件共享


【桌面】

优化 应用定位的动画效果


【主题】

新增 支持赠送主题


【相机】

优化 录像质量


【图库】

新增 云相册支持视频同步

新增 云相册支持原图下载

优化 云相册自动上传逻辑

修复 某些情况下，下载错误文件导致目录不显示的问题

修复 重启后屏保失效的问题


【音乐】

优化 “在线”标签增加引导气泡


【天气】

修复 某些情况下，桌面天气图标不显示的问题


【文件管理器】

新增 支持访问Samba网络文件共享（小米路由和其他Samba服务器都可以访问）


【浏览器】

新增 通知栏快速搜索


【日历】

优化 新版日历支持米1及第三方机型


【视频】

新增 支持视频离线（目前仅支持部分源）

新增 DLNA视频列表可按拼音字母快速查找

新增 乐视、百视通视频资源

新增 Dirac音效菜单

优化 合并"视频播放器"与"视频"应用

修复 某些情况下，连续换台时声音不消失的问题

修复 播放云相册中的视频时显示“本地”的问题

修复 电视直播页面，按一次返回键就退出界面的问题


【小米帐户】

修复 云服务头像模糊的问题


【游戏中心】

修复 省流量模式下，仍然下载图片的问题

修复 “在线客服”中链接无法点击的问题


【安全中心】

优化 “占用内存”的应用勾选状态，与系统长按桌面键的“近期任务”应用锁定状态保持一致
