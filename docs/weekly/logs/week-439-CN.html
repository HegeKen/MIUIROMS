MIUI 第439周发布公告及更新日志

原链接：MIUI 第439周发布公告及更新日志

▍发布时间

MIUI11 开发版9.11.28 11:00发布，部分机型延迟发布


▍升级说明

  1. MIUI 11开发版可进入“设置—>我的设备—>MIUI版本”在线升级，升级前务必备份重要数据
  2. 部分机型可能因功能进版时间略有差异，具体功能请以手机端展示日志为准

▍重要说明

  1. 小米 8 青春版、小米 MAX3、小米CC 9、小米CC 9 美图定制版、Redmi Note 8 Pro 启动Android 10升级，本周暂停发布
  2. Redmi Note 7、Redmi Note7 Pro 因测试系统稳定性，公测发布延迟
  3. 小米8 SE、小米9 Pro 5G、Redmi Note 8 因解决影响稳定性Bug，本周暂停发布
  4. 米Note 4X 、红米4X、红米Note 5A标准版、红米Note 5A高配版、小米5C、小米Max 2、小米Note 2、小米MIX、小米手机5s、小米手机5s Plus 、红米5 Plus、小米5X、红米5A、红米5、小米平板3 正式停止开发版公测
  5. 由于不在MIUI11适配计划机型内，小米平板4/4Plus暂停更新

▍更新日志


系统

新增 未知来源安装应用时支持单次允许安装

新增 应用安装完成支持前往设置应用权限

新增 安装应用时显示版本升级信息

新增 安装应用时新增覆盖安装提示

新增 应用包管理组件适配深色模式

新增 应用包管理组件适配MIUI 11视觉

优化 在锁屏闹钟页面屏蔽全面屏手势的功能（Redmi Note 8 Pro、小米6X、小米手机8SE、小米手机8透明探索版）

优化 应用安装失败时删除安装包的逻辑

修复 在桌面解锁时偶现的黑屏闪烁问题

更新谷歌安全补丁（2019#11），提升系统安全性


锁屏、状态栏、通知栏

新增 原生媒体通知样式（红米手机6Pro 、小米手机6 ）

优化 允许正在运行的自定义媒体通知在锁屏上显示（红米手机6Pro 、小米手机6 ）

修复 手机分身中，无法通过通知栏进入应用通知设置页的问题（红米手机6Pro 、小米手机6 ）

修复 个别应用的通知内容无法显示完整的问题（红米手机6Pro 、小米手机6 ）

修复 自定义快捷开关点击无反应、反应慢、首次打开不可点击等问题（红米手机6Pro 、小米手机6 ）


相机

新增 专业模式光焦分离、专业拍照功能介绍（小米8SE、Redmi Note 8 Pro）

优化 专业模式参数名称（小米8SE、Redmi Note 8 Pro）


游戏加速

修复 点击应用悬浮窗全屏按钮后应用闪退的问题
