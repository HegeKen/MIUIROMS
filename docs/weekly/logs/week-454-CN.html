MIUI 第454周发布公告及更新日志

社区链接：MIUI 第454周发布公告及更新日志

▍发布时间和机型

MIUI12 开发版20.5.21 12:00发布，部分机型延迟发布


▍升级说明

  1. MIUI 开发版可进入“设置—>我的设备—>MIUI版本”在线升级，升级前务必备份重要数据
  2. 部分机型可能因功能进版时间略有差异，具体功能请以手机端展示日志为准

▍重要说明

  1. Redmi Note 7、Redmi Note7 Pro、Redmi Note 8 Pro、小米9 因解决影响稳定性Bug，本周公测发布延迟
  2. 小米10 Pro、小米10、Redmi K30 Pro 因出现打电话无法黑屏问题，正在测试稳定性，预计下周一发布
  3. Redmi Note 8、小米CC 9e 适配Android 10升级，停版一段时间
  4. 2020年6月20日起，以下机型将停止开发版内测公测：小米6、红米Note5、小米6X、小米Note3、小米MIX2

▍更新日志


【控制中心】

新增 桌面左侧下滑显示通知中心，右侧下滑显示控制中心

新增 在通知中心右上角状态栏下拉，可以呼出控制中心

修复 在新版控制中心编辑开关后，旧版部分快捷开关消失的问题

修复 第一次下拉控制中心后，亮度条无法使用的问题

修复 进入控制中心不显示toast的问题


【深色模式】

优化 “深色模式下调暗壁纸”功能，增加调暗后壁纸的明度和色彩


【系统】

优化 更新谷歌安全补丁（2020#05），提升系统安全性


【状态栏、通知栏】

新增 悬浮通知背景样式使用柔化毛玻璃效果（小米手机9、小米10 Pro）

修复 下拉通知栏偶现的显示卡顿问题


【相机】

新增 专业模式光焦分离、专业拍照功能介绍

新增 全新相机界面，全新交互体验和动效设计

新增 全面屏拍摄，支持首屏功能自定义(最少2个，最多10个)、音效自定义(4套音效)、颜色自定义(5套配色)

新增 声控拍照功能 使用 "拍照"、"茄子"实现声控拍照（Redmi K30 5G）

新增 录像过程中变焦轴操作快速变焦

新增 支持拍照模式下长按拍照按钮进行录像操作

优化 专业模式参数名称

优化 Menu平铺样式、分辨率/画幅选择操作更便捷

优化 相机快速变焦轴、一键快速变焦（Redmi K30 5G）

优化 动态照片开关打开后功能常驻（Redmi K30 5G）


【系统更新】

系统更新界面重大改版，尤其在下一次检测到新版本时，您将体验到丝滑且奇妙的动画体验，敬请期待


【三方应用异常分析】

新增三方应用异常分析功能，提示您部分闪退问题的原因与解决方法。更多功能开发中，期待您的反馈（小米手机9 ）


【省电优化】

新增 自动任务支持备份还原

新增 省电模式可选优化项“息屏显示”

修复 耗电统计中唤醒统计异常
