MIUI 第480周发布公告及更新日志

社区链接：MIUI 第480周发布公告及更新日志

▍发布时间和机型

MIUI12 开发版20.12.9版本 13:00前发布，部分机型已发布


▍升级说明

  1. MIUI 开发版可进入“设置—>我的设备—>MIUI版本”在线升级，升级前务必备份重要数据
  2. 部分机型可能因功能进版时间略有差异，具体功能请以手机端展示日志为准

▍重要说明

  1. 为迎接MIUI12.5版本发布，12.14日下周开始将停版一段时间，永远相信美好的事情即将发生
  2. 因项目工程需要，Redmi 10X、Redmi 10X Pro、Redmi K30 至尊纪念版 会推迟至12月18日开始停版，感谢大家的理解（如有变动会第一时间同步大家）
  3. 由于Android大版本升级，Redmi Note 8、Redmi Note 8pro、Redmi K20 Pro、Redmi K30S 至尊纪念版、小米CC9、小米CC9 美图定制版、小米9 SE、小米9、小米9Pro 发布最后一个基于Android 10的20.12.9版本，感谢大家的理解
  4. 今日预计13点前发布，部分机型已发布，感谢大家的理解
  5. 由于特定情况下出现重启问题，小米CC9 本周暂停发布，感谢大家的理解
