MIUI 第517周发布公告及更新日志

更新预告：MIUI开发版第517周更新预告，请注意查收

社区链接：MIUI 第517周发布公告及更新日志

▍发布时间和机型

MIUI 12.5 开发版 21.10.13 版本 今天陆续发布，部分机型已发布


▍升级说明

  1. MIUI 开发版可进入“设置—>我的设备—>MIUI版本”在线升级，升级前务必备份重要数据
  2. 部分机型可能因功能进版时间略有差异，具体功能请以手机端展示日志为准

▍重要说明

  1. Xiaomi Civi 首次发布开发版, 多项优化与改进, 向勇敢尝鲜的米粉们致敬
  2. 由于Android大版本升级，小米11 青春版、Redmi K40、小米10S 2021年9月22日起暂停发布公测，在内测周度参与标准中，大家可对正在使用的版本投票和发布体验贴，感谢大家的理解
  3. 开发版内测版本问题逐步修复后，会陆续推送开发版公测版本，安卓12已知问题，点此查看>>
  4. 2021年11月27日起，以下机型将停止开发版内测公测：小米9、小米9SE、Redmi K20 Pro、Redmi K20、小米CC9 美图定制版、Redmi Note 8 Pro、小米CC9、Redmi Note 8、小米9 Pro

▍更新日志


【系统】

修复使用小窗等操作后，应用方向错误的问题
修复连接需要登录认证的WiFi网络后，HTML查看器不转到登录认证页面的问题
修复设置页小米帐号头像刷新不及时的问题

【音乐】

新增下一首播放功能，不错过每一首不经意间的好声音
新增桌面歌词功能，体会每一句歌词在屏幕间的流动
优化隐私及数据合规相关功能，保护米粉们的数据安全

【短信】

优化部分体验问题

【浏览器】

新增密码管理，提升使用便利

【小米天气】

优化虚拟按键背景色适配
修复无数据时小部件编辑页白屏问题
修复小部件编辑页跳转错误问题
